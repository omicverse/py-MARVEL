# Phase 3 Plate R References

This directory is the committed reference root for Phase 3 plate plotting, PCA, modality, and iso-switch work.

The canonical contract is the manifest generated by `scripts/export_phase3_plate_reference.R`.

- `run_pca_gene_coords.tsv`, `run_pca_gene_explained.tsv`, `run_pca_splicing_coords.tsv`, and `run_pca_splicing_explained.tsv` come from the shared tiny plate fixture.
- `plot_values_gene.tsv` and `plot_de_gene_global.tsv` are exported as stable tables for exact Python parity.
- `prop_modality.tsv` comes from the shared tiny plate fixture and intentionally preserves the upstream R `NA` modality drift.
- `modality_change.tsv`, `iso_switch.tsv`, and `iso_switch_plot_expr.tsv` come from `marvel.demo.rds`, because these public APIs depend on richer differential splicing state than the tiny shared fixture contains.
